# 高専祭　会計スクリプトver2

### システム用件  
OS: Windows  
プログラム: Python3.10  
ブラウザ: Google Chrome  
  
### インストール  
「setup.bat」を実行  
「start_system.bat」を実行  
会計する  
pipパッケージをアンインストールしたいときは「uninstall.bat」を実行
